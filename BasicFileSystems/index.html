RHCSA Objectives:
  • View, list, create, & delete data partitions on MBR & GPT disks
  • Create, mount, and unmount various file systems
  • Configure rhel to mount file systems at boot
  • Safely add partitions, logical volumes, and swap volumes to a system

File Systems

For modern operating systems to use hard drives, they must be formatted with partitions and a file system.

Unlike some OSes, Linux typically uses multiple partitions. Some reasons for this on Linux:
  • Securing Data (separating /home/ from / & /etc)
  • Perfomance (use drives in parallel)
  • Back-Ups (isolate relevant parts of the OS to back up)
  • Fail-safe (one partition filling won't necessarily make other partitions unreachable)
Some Filesystem Commands
  • blkid = Description of filesystems that are mounted
  • lsof = List open files and processes in a directory
  • lsblk = List info about block devices (drives) accessible by the system
  • df = Displays info about filesystems on the system
  • tune2fs = show & manage file system properties
